## @import
If you want to use contents from this repo yourself, use the jsDelivr Syntax:
```bash
@import url('https://cdn.jsdelivr.net/gh/plorii/jellyfin-css@main/<file>')
```
Replace `<file>` with the file you want. For example: file `jellyfin-drawn-icons8.png` in subfolder `icons`: 
```bash
@imprt url('https://cdn.jsdelivr.net/gh/plorii/jellyfin-css@main/icons/jellyfin-drawn-icons8.png')
```
